Commenting System

This Commenting System is made with :
  • 1) HTML5
  • 2) CSS
  • 3) BOOTSTRAP4
  • 4) ANGULAR CLI
  • 5) JSON

Comment are taken as input and are displayed as soon as they have been posted.

Data in form of JSON has been used to stored the details of the commnets.

Few features:

Only the user that has commented the comment can delete it.

Reply functionality has been implemented.

With power of local storage particular user view from will remain same even on refresh.

Like functionality has also been implemented user can like the comment with numbers of likes in display.

More features that can be implemented:
